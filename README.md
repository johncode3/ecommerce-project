# TechHub – E-Commerce Website

## Overview

TechHub is a simple e-commerce website created as a university assignment.
The goal of this project is to practice front-end web development using HTML, CSS, and JavaScript while building a multi-page online store.

This project will also serve as a foundation for future improvements such as connecting to a database and implementing backend technologies.

## Purpose

* Built as a school assignment.
* Practice front-end web development.
* Learn how a basic e-commerce website structure works.
* Prepare for future backend integration using technologies like PHP or Django with databases such as MySQL, Oracle, or SQL Server.

## Technologies Used

* HTML
* CSS
* JavaScript

Future technologies that may be added:

* PHP or Django (for backend)
* MySQL / Oracle / SQL Server (for database)
* React or Tailwind CSS (for modern UI)

## Website Pages

This project contains multiple pages to simulate a basic online shopping experience:

* `homepage.html` – Main landing page
* `product.html` – Product listing page
* `iphone_17_pro_max.html` – Product detail page
* `cart.html` – Shopping cart page
* `checkout.html` – Checkout page
* `login.html` – User login page
* `register.html` – User registration page
* `about.html` – Information about the website
* `contact.html` – Contact page
